PerlChina Advent
===========

## Dependencies 依赖软件

 * aspell

## INSTALL 安装指南

如果你还没有安装 [cpanm](https://metacpan.org/pod/cpanm)：

 * curl -L http://cpanmin.us | perl - App::cpanminus

完成后

 * cpanm --installdeps .
 * perl script/perl_china_advent daemon

然后使用浏览器访问 <http://127.0.0.1:3000/>
